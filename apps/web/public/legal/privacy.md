# 《隐私政策》(Privacy Policy) - h1bfinder.com

**生效日期：2026年3月9日**

h1bfinder.com 致力于保护您的个人隐私。作为 AI 导向的数据源平台，我们坚持最小化收集原则。

## 1. 我们收集的信息
*   **API 请求日志**：当您使用 `h1b-finder` 技能或访问网站时，我们会记录 IP 地址、请求的时间、请求的参数。这仅用于性能优化和防御恶意攻击。
*   **搜索习惯**（匿名）：我们会记录热门搜索词（如：某城市、某公司），以改进 AI 推荐逻辑。
*   **Cookies**：我们可能使用少量 Cookies 来记住您的语言偏好（如：中英文切换）。

## 2. 我们不收集的信息
*   **身份信息**：除非您主动联系我们，否则我们不收集您的姓名、电话、地址等真实身份信息。
*   **本地私有数据**：`h1b-finder` 技能在您的 OpenClaw 工作空间运行，我们无法访问您的本地文件、对话历史或工作私密信息。

## 3. 数据处理与存储
您的 API 请求数据通常存储在美国境内服务器上。我们不会将您的搜索日志出售给任何第三方（如猎头公司或广告商）。

## 4. 第三方服务
我们使用 Google Analytics 来统计流量。您可以查阅 Google 的隐私政策了解详情。

## 5. 您的权利
如果您需要删除您的 API 请求记录，或有任何隐私疑虑，请通过 [contact@h1bfinder.com] 与我们联系。
